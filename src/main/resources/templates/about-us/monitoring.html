Інформація про моніторинг

Опитування

  • Анкетування здобувачів освіти щодо організації дистанційного навчання
  • Опитування заінтересованих осіб щодо якості освітньо-професійної програми підготовки фахівців за освітньо-професійним ступенем «Фаховий молодший бакалавр»
  • Опитування здобувачів освіти щодо організації освітнього процесу та рівня задоволеності матеріально-технічним забезпеченням
  • Анкета «Вільний вибір навчальних дисциплін здобувачами освіти»
  • Анкета для здобувачів освіти щодо форм контрольних заходів та критеріїв оцінювання
  • Анкета для педагогічних працівників щодо організації дистанційної роботи
  • Анкета щодо втілення ідеї студентоцентризму в освітній процес
  • Анкета для випускників щодо організації освітнього процесу в коледжі
  • Анкета-опитувальник для випускників, щодо якості освітньо-професійної програми
  • Опитувальник для роботодавців
  • Анкета для опитування стейкхолдерів, щодо якості освітньо-професійної програми
